# 패스워드 변경

## Background

사용자 계정의 보안을 유지하려면 패스워드를 주기적으로 변경하거나 유출 의심 시
즉시 변경할 수 있어야 합니다. 현재 시스템에는 패스워드 변경 기능이 없어 보안
취약점이 존재합니다.

## Goal

로그인한 사용자가 기존 패스워드를 확인한 후 새 패스워드로 안전하게 변경할 수
있도록 합니다. 이를 통해 계정 보안을 강화하고 패스워드 유출 시 신속한 대응이
가능합니다.

## How it works

사용자는 PATCH `/users/{username}/password` 엔드포인트로 패스워드를 변경합니다.

### 인증된 사용자

- 기존 패스워드와 새 패스워드를 요청에 포함합니다
- 시스템이 기존 패스워드를 검증합니다
- 검증 성공 시 새 패스워드로 변경되고 200 응답을 받습니다
- 검증 실패 시 422 응답을 받고 패스워드는 변경되지 않습니다

### 인증되지 않은 사용자

- 401 Unauthorized 응답을 받습니다
- 패스워드 변경이 불가능합니다
